# TONY - Gesture Music Controller

손 제스처로 실시간 음악을 제어하는 시스템

## 설치

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 실행

```bash
python main.py test_music.mp3
```

## 제어 방법

- **✊ 주먹**: 재생/정지
- **🤏 집게**: 볼륨 조절
- **✌️ 2개**: 딜레이 ON/OFF
- **🤟 3개**: 코러스 ON/OFF
- **🎚️ 좌우**: 팬 제어
- **Q**: 종료
